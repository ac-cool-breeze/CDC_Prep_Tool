# USAF_CDC_Prep_Tool
Ingests UREs and Answers, creates random test with associating answers.

Takes .txt files in the following forms, note the '.' separating the numbers and content:

Questions:
----------------------------
1. some question

a. answer
b. answer
c. answer
d. answer

2. some question

a. answer
b. answer
c. answer
d. answer
-----------------------------

Answers:
-----------------------------
1. a
2. a
3. c
4. d
-----------------------------
